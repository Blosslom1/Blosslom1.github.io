编辑器使用的是Hbuilder，所有快捷键均以Hbuilder为标准

HTML

对文本注释Ctrl+？

<!DOCTYPE html>声明标签为html界面

HEDE

用于声明文本的编码格式，还有一些其他的格式例如gbk等等。

网站的关键字，用于百度收录

网站的描述，用于百度收录

作者信息

重定向的位置，1表示1秒

定义网站上方的图标

css样式的文件

标题

标签的大小从1到6依次递减

水平线

画一个水平线，自结束width表示水平线的粗细

换行

强制换行标签

空格

&nbsp;

段落

段落，下一段就会换行

字体

里面的文字变成斜体

字体加上下划线

字体加粗

删除线

字体的格式可以叠加

列表

无序列表，列表前面是用点表示的

有序列表，列表前面是用数字来表示，type表示使用的是罗马数字

自定义列表，dt是列表的名词，dd是对列表的解释说明。

图片(img)

链接（a）

target包括(_blank，_parent，_self，_top)

href可以是一个网站也可以是一个锚点，锚点意思是id等标识

字节标签（span）

用来包裹部分文字，让页面符合规范。

表单（form）

表单用来提交内容到服务器

method有两个方法一个get一个post，get是http传输post是https传输

音频

audio音乐，video视频。

src是路径，controls表示列出控件，autoplay自动播放，loop重复播放

CSS

样式

内联样式

内部样式

外部样式

在正规样式编辑中主要是使用外部样式

选择器

执行顺序是从上向下，会以最后一个执行的为标准。

类选择器（class）

在html元素中使用class命名一个类，可以使用多个名字，元素之间的class可以重名。

在css文件中进行选择，使用.表示是class选择器。

id选择器

在html元素中使用id命名，具有唯一性。

在css文件中使用#选择

群组选择器

在css中同时选中多个，进行打组改变样式，使用，隔开

包含选择器

父子关系的标签，使用空格隔开表示父子关系

伪类选择器

hover

鼠标滑过效果。选择器:hover{属性}

通配符选择器（*）

主要用于这个页面的归零

权重大小

  • 元素选择器：0001
  • calss选择器：0010
  • id选择器：0100
  • 伪类选择器：0010
  • 内联选择器：1000
  • 包含选择器：选择器权重之和
  • 群组选择器：单独计算，元素权重不相加
  • 权重相同听后面的
